建立ssh互信使用方法:
在当前用户下，执行脚本：
./crtSSH.sh node01 node02 node03 node04

注：  node01，node02，node03，node04 为/etc/hosts 机器节点名称

-----------------------------------------------------------------------------
删除ssh互信使用方法:
在当前用户下，执行脚本：
./cleanSSH.sh node01 node02 node03 node04

注：  node01，node02，node03，node04 为/etc/hosts 机器节点名称
